Translate Table Key

Trans Text
==========

x - None
*1, *2, *3, *4 - Normal
*e, *f, *g, *h - Normal
*6 - Comma	*a - Empty String
*7 - Line Token	*b - Current Scope
*8 - Space	*c - Empty String
*9 - Module 	*d - ::
*i - Clear Trans Text


Flags
=====

x - Don't Care
r - Repeat
y - Finish Processing
z - Optional Token
