UML Work Bench Readme
=====================

1) Source code is assumed to have the correct syntax.

2) Default values for function arguments - NYI.

3) Interface display is not part of the UML DI bounding rectangle.

4) Classes, interfaces and enums can not have the same name.

5) { for a function, class or interface must be end the end of a line,
   or on the next line on its own.
